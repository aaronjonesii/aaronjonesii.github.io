NgRx authentication template. Using this post as a building block to build my official WebApp.

## Authentication Workflow
##### `Not Authenticated`
1. Get User
2. Not Authenticated
3. Login
4. Authenticated
5. Check Token
6. Login Success
7. Get User Info

##### `Authenticated`
1. Get User
2. Authenticated
3. Check Token
4. Login Success
5. Get User Info
    
##### Testing
1. Authenticate
2. Authentication Success
3. Get User Info
